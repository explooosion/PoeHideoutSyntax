## Features

This is a POE ( Path of Exile ) hideout syntax for [VS Code](https://code.visualstudio.com/).

The extension will make your `.hideout` more visible and editable.

## Setting

1. Open your filter file
2. Change language Mode ( `Ctrl`+`K`, `M` )
3. Choose Filter
